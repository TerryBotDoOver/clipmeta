# ClipMeta

**AI-powered stock footage metadata generator** — automatically generate titles, descriptions, and keywords from your video clips.

🔗 **[clipmeta.app](https://clipmeta.app)**

---

## What it does

ClipMeta watches your actual video frames and generates platform-ready metadata:
- **Titles** — SEO-optimized, varied, never "Aerial View of..." on every clip
- **Descriptions** — natural language, buyer-intent focused
- **Keywords** — 15–50 per clip, strongest first, no generic filler
- **Categories** — auto-matched to each platform's taxonomy

## Platform support

Export CSV files in the exact format each platform expects:

| Platform | Keywords | Format |
|----------|----------|--------|
| Blackbox.global | Up to 49 | CSV with categories |
| Pond5 | Up to 50 | CSV with location fields |
| Adobe Stock | Up to 45 | CSV with numeric categories |
| Shutterstock | Up to 50 | CSV with editorial flags |
